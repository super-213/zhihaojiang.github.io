总结

  • 树模型 (XGBoost / LightGBM / 随机森林) → 表现最好，准确率 0.91，ROC-AUC 高，Precision 接近 1，但 Recall 约 0.78–0.79，有一定漏判风险。
  • 朴素贝叶斯 → 简单快速，但 Recall 下降到 0.70，漏判更多。
  • KNN → 表现最差，不适合该任务。

结论： 金融场景推荐使用 XGBoost / LightGBM / 随机森林，并可通过 阈值调整、类别权重、SMOTE 等方法进一步优化 Recall。
