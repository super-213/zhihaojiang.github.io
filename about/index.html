智能停车场管理系统（car_parking_system）

  • 时间: 2024.12.09
  • 描述: 基于 Python、MySQL 和 NetworkX 构建的智能停车场管理系统，灵感来源于学习 Python 与 SQL 后对智能交通的兴趣探索。
  • 链接: https://github.com/super-213/car_parking_system
  • 说明:
    • 在学习 Python 与 SQL 后，发现传统停车场管理效率低，难以智能匹配车位，存在用户体验差的问题。
    • 因此，我的目标是构建一个可智能推荐车位与路径的系统，提升车位利用率与用户停车效率。
    • 使用 Python 实现车牌入出库记录的自动化管理，实时更新数据库中车位状态。利用 MySQL 管理车位与车辆信息，支持事务处理，增强系统稳定性与数据一致性。基于车辆类型设计推荐算法，结合 NetworkX 构建场内图结构并计算最短寻车路径。
    • 实现车位精准匹配与智能路径导航，系统支持扩展至多车场环境，增强用户体验并具备推广价值。
  • 技术栈: Python, MySQL

  • 贡献:
